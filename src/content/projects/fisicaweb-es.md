Una aplicación web educativa diseñada para enseñar conceptos de física a través de simulaciones interactivas y visualizaciones. Desarrollada para un proyecto universitario, combina React para componentes de interfaz dinámica con Astro para optimizar la entrega de contenido estático. Incluye herramientas interactivas de resolución de problemas para ayudar a los estudiantes a comprender principios físicos complejos.
