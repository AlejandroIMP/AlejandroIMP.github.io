Una aplicación de e-commerce sofisticada que presenta un catálogo completo de productos, búsqueda inteligente y filtrado por categorías, además de un carrito de compras interactivo con actualizaciones en tiempo real. Construida utilizado React para la gestión dinámica del estado y Tailwind CSS para un estilo elegante y responsive. La aplicación se integra perfectamente con la API Fake Store de Platzi, demostrando mejores prácticas en consumo de APIs y manejo de datos. Optimizada para la experiencia del usuario con navegación fluida y descubrimiento intuitivo de productos.
